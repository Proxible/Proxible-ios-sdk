filterBeacons:withProximities:accuracy:getClosest:

Returns a Array with beacons filtered on the given list of proximities and accuracy

+ (NSArray *)filterBeacons:(NSArray *)beacons withProximities:(NSArray *)proximities accuracy:(CLLocationAccuracy)accuracy getClosest:(BOOL)closest

Parameters

beacons

An array of beacons to filter

proximities

The list of proximities a beacon has to be in

accuracy

The accuracy of the beacon proximity from the mobile device

closest

A boolean indicating if only the closest beacon should be returend. If NO all beacons mathing the filter criteria are returned. If YES of all the beacons mathing the filter criteria only the closest of them all is returned. The beacons accucary is used to determine the closest beacon.

Return Value

Array of beacons that match the filter criteria.
